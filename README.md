# Password Manager (CLI)
**Автор:** Клочков Денис

### Описание
Консольное приложение для безопасного управления паролями. Позволяет хранить учетные данные, удалять их и генерировать сложные пароли.

### Запуск
1. Установите Python 3.x.
2. Склонируйте репозиторий.
3. Запустите: `python main.py`.

### Функции
- Хранение данных в формате JSON.
- Генерация паролей любой длины.
- Валидация вводимых данных.
